Open Source

Το client-side κομμάτι του confessions είναι open source!

Για να μπορέσουμε να κερδίσουμε την εμπιστοσύνη του κοινού μας, αποφασίσαμε να κάνουμε το client-side κομμάτι (client front, client back) λογισμικό ανοιχτού κώδικα! Αμα είστε προγραμματιστής και γνωρίζετε GoLang σας προτρέπουμε να δείτε το project μας στο GitHub και να μας πειτε την γνώμη σας!

github.com/enderian/confessions

Το λογισμικό που μας βοηθάει:

  • AngularJS (Handles our single-page landing webpage)
  • Bootstrap (Provides their beautiful theme and components)
  • jQuery (Lightweight DOM manipulation and AJAX)
  • toastr (delivers notifications)
  • underscore.js (powerful data manipulations)
  • webpack (compiles our JavaScript and CSS into a single file)
